# Runtime: Video Composition

Video composition is good at timed visual events, scene sequence, motion rhythm,
and typography that changes over time.

Use this runtime when the visual idea needs time:

- a title being pressed by a cost meter;
- a token path draining through a level map;
- a diagram revealing one step at a time;
- a material transition such as tearing, peeling, stamping, or accumulating.

## What This Runtime Is Good At

- Sequential explanation.
- Motion as argument.
- Timed hierarchy changes.
- Scene transitions.
- Audio-reactive or narration-synced emphasis.

## What This Runtime Is Bad At

- Exact long-form reading surfaces.
- Static print deliverables.
- Single-frame proof screenshots.
- Motion added only for decoration.

## Compile a Score into a Video Spec

Use these slots:

```text
Output:

Forbidden:
```

## Rule

Do not turn a still image into a video by adding camera drift.

Name the visual event that time makes possible.
