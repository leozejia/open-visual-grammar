# Runtimes

A runtime is where the score executes.

Current runtime notes:

| Runtime | File | Good at |
| --- | --- | --- |
| Image generation | `image-generation.md` | Mood, texture, metaphor, broad composition |
| Deterministic layout | `deterministic-layout.md` | Exact text, exact hierarchy, exact dimensions, repeatable output |
| Font packaging | `font-packaging.md` | Packaged fonts, license metadata, host fallback policy, local/CI/production parity |
| Shader | `shader.md` | Continuous fields, light, material, motion, invisible systems |
| Video composition | `video-composition.md` | Scene sequence, visual events, motion rhythm, timed typography |

Future runtimes may include slide decks and game engines.
