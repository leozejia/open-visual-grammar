# Runtime: Shader

Shaders are good at continuous fields, light, material, motion, and invisible
systems becoming visible.

Use a shader runtime when the visual idea depends on behavior over time:

- cache heat;
- flowing token streams;
- memory decay;
- routing fields;
- signal interference;
- pressure or accumulation.

## Compile a score into a shader spec

Use these slots:

```text
Field:
Forces:
Motion:
Color temperature:
Density:
Interaction:
Time behavior:
Readable focus:
Forbidden:
```

## Rule

Do not start from colors or effects.

Start from the invisible system the shader should reveal.
