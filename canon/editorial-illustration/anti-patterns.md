# Editorial Illustration Anti-Patterns

## Generic AI symbolism

Robots, glowing brains, circuit halos, and floating network nodes usually make
the image less specific.

## Mood without argument

Atmosphere is not enough. The scene should carry a point.

## Mixed metaphor

A person, maze, storm, dashboard, and broken chain in one image is not richer.
It is less readable.

## Text dependency

If the image only works because the title explains it, the metaphor is weak.
