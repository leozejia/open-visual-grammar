# Metaphor Distance

Metaphor distance controls how far the image moves away from the literal topic.

## Near metaphor

Use for beginners, tutorials, and operational docs.

Examples:

- session handoff as a document package;
- token cost as a ledger;
- cache state as a layered storage shelf;
- screenshot proof as a real interface frame.

## Medium metaphor

Use when the reader understands the topic but needs a stronger image.

Examples:

- context as a workspace that gets dusty;
- output cost as heat;
- model routing as a rail switch;
- prompt drift as a compass losing north.

## Far metaphor

Use for essays, brand films, and atmospheric work.

Examples:

- context as weather;
- model selection as theater casting;
- cache as geology;
- agent handoff as ritual.

## Decision

When in doubt, move closer.

If the visual needs a paragraph of explanation, the metaphor is probably too
far for a cover or hero image.
