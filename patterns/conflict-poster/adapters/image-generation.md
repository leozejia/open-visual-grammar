# Image Generation Adapter

Use this adapter when compiling `conflict-poster` into an image prompt.

## Required slots

```text
Output:
Audience:
Conflict pair:
Title:
Main metaphor:
Composition model:
Information density:
Emotional temperature:
Safe area:
Forbidden:
```

## Rule

If exact title typography matters, generate the visual field first and typeset
the title deterministically.

Do not turn the poster into a dashboard, pricing table, or multi-panel
infographic.
