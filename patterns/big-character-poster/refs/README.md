# Reference Images / 参考图

These images are operator-selected references for the pattern. They show visual
identity, not a fixed prompt recipe.

这些图片是经过人工评选的范式参考图。它们说明视觉气质，不是固定 prompt。

## `agent-runtime-cost-x-cover.png`

Why it belongs:

- strong first read;
- public argument rather than neutral tech banner;
- concrete cost/runtime object;
- rough poster energy with direct attitude.

保留原因：

- 第一眼强；
- 是公共论点，不是中性技术 banner；
- 有具体成本和 runtime 证据物件；
- 粗粝海报感和态度成立。

## `context-token-economics-x-cover.png`

Why it belongs:

- context/cost conflict is visible before reading;

- enough detail for inspection without weakening the first read.

保留原因：

- 上下文和成本冲突在阅读前可见；
