# Flowing Gaze Minimal Cover

Original name: `流动的凝视-极简封面`

Status: seed

## Invariants

- one subject or one viewpoint;
- large negative space;
- quiet narrative tension;
- retro minimal cover logic;
- pen drawing or etching texture;
- restrained palette.

## Recommended Cases

- personal essays;
- abstract ideas;
- reflective covers;
- topics that need pause rather than urgency.

## Weak Fit

- step-by-step tutorials;
- high-density information;
- hot public debate;
- product launch visuals.

## Avoid

- empty whitespace without tension;
- generic SaaS panels;
- loud gradients;
- decorative glow;
- random text, logos, signatures, or watermarks.
