# Flowing Gaze Minimal Cover

中文名：流动的凝视·极简封面
English name: Flowing Gaze Minimal Cover

Status: seed

## Definition / 定义

EN: Flowing Gaze Minimal Cover is a quiet cover pattern built around one subject,
large negative space, and a subtle viewpoint that makes the reader pause.

CN: 流动的凝视·极简封面是一种安静的封面范式，依靠单一主体、大留白和带叙事感的视角，让读者停顿一下。

## Invariants / 不变量

EN:

- one subject or one viewpoint;
- large negative space;
- quiet narrative tension;
- retro minimal cover logic;
- pen drawing or etching texture;
- restrained palette.

CN:

- 一个主体或一个视角；
- 大面积负空间；
- 安静的叙事张力；
- 复古极简封面逻辑；
- 钢笔、蚀刻或细线质感；
- 克制配色。

## Recommended Cases / 推荐场景

EN:

- personal essays;
- abstract ideas;
- reflective covers;
- topics that need pause rather than urgency.

CN:

- 个人化文章；
- 抽象概念；
- 反思型封面；
- 需要停顿感而不是紧迫感的话题。

## Weak Fit / 不适合

EN:

- step-by-step tutorials;
- high-density information;
- hot public debate;
- product launch visuals.

CN:

- 步骤教程；
- 高密度信息表达；
- 强公共争论；
- 产品发布视觉。

## Avoid / 避免

EN:

- empty whitespace without tension;
- generic SaaS panels;
- loud gradients;
- decorative glow;
- random text, logos, signatures, or watermarks.

CN:

- 没有张力的空白；
- 通用 SaaS 面板；
- 大声的渐变；
- 装饰性发光；
- 随机文字、logo、签名或水印。
