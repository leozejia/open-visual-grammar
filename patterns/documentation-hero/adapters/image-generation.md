# Image Generation Adapter

Use this adapter when compiling `documentation-hero` into an image prompt.

## Required slots

```text
Output:
Reader level:
Operational concept:
Central system metaphor:
Composition model:
Information density:
Emotional temperature:
Text handling:
Forbidden:
```

## Rule

The generated image should support trust and comprehension. It should not
pretend to be exact UI proof.

If the page needs exact labels, UI state, or diagrams, use deterministic layout
or real screenshots instead.
