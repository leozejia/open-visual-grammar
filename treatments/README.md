# Treatments

Treatments are surface-handling systems.

They are not canons.

A canon answers the visual problem. A treatment adjusts the surface after the
canon is chosen:

```text
canon -> score -> optional treatment -> runtime spec
```

Examples of treatment decisions:

- paper texture vs glass surface;
- pixel finish vs hand-drawn line;
- low-saturation ink vs bright poster contrast;
- journal collage vs clean geometric reduction.

## Current migrated treatments

| Treatment | Notes |
| --- | --- |
| Eastern textured hand-drawing | Paper texture, relaxed lines, low saturation, metaphor-friendly. |
| Retro pixel | 8-bit or 16-bit pixel language, useful for geek culture and retro tech. |
| Narrative journal infographic | Hand-drawn timelines, comparisons, and structured notes. |
| Whimsical journal sketch | Watercolor, marker texture, collage, lifestyle or idea diary tone. |
| Flowing minimal gaze | Retro minimal, negative space, pen or etching feel. |
| Elegant simple hand-drawing | Refined line work, warm web illustration feel. |
| Minimal line sketch | Clean lines, restrained contrast, low saturation, lots of whitespace. |

## Rule

Do not start from a treatment.

Choose a canon first. Apply one treatment only if it strengthens the score.

If a treatment develops its own visual problem, references, composition rules,
and anti-patterns, promote it into a canon.
